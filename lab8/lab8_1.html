Свойства CSS

CSS (Cascading Style Sheets) — это язык стилей, используемый для описания внешнего вида документа, написанного на HTML. С помощью CSS можно управлять цветом, шрифтами, отступами, размерами и другими аспектами оформления веб-страниц.

Некоторые основные свойства CSS:

  • color - задает цвет текста.
  • font-family - определяет шрифт текста.
  • margin - устанавливает внешние отступы.
  • padding - задает внутренние отступы.
  • background-color - определяет цвет фона элемента.
